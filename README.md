# ⌨️ Typing Test App

This project is inspired by Monkeytype 🚀 and was built to combine two things I enjoy: practicing my typing skills daily and improving my development skills 💻.

I’m creating this app as a personal project to sharpen my programming knowledge, explore new technologies ⚡, and learn more throughout the development process 📚.

## 🛠️ Tech Stack

- ⚛️ Next.js
- React
- Redux
- Tailwind CSS 🎨
- C#
- .NET Core
